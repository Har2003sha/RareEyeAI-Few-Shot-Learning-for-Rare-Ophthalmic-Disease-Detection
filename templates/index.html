Why Few-Shot Learning?

Rare ophthalmic diseases, by definition, have very few labelled patient images available for training. Traditional deep learning needs thousands of examples per class. Prototypical Networks instead learn a general-purpose embedding space where each class is represented by a single "prototype" — the mean embedding of a handful of support examples — enabling reliable classification from as few as 1–5 images per rare disease class.

This deployment ships with a procedurally generated demo support set so the full pipeline runs out-of-the-box. See the Support Set page (after login) and the project README for instructions on plugging in a real, clinically-sourced rare-disease dataset.
